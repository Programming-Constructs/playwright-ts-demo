# playwright-ts-demo
Setup demo framework to test multiple projects 

# Prerequisites
- Node.js
- npm
- Typescript

# Setup
- npm install
- npx playwright install

# Run All tests
- npx playwright test 

# Run Specific spec
- npx playwright test ./projects/playwright-dev/docs-page/docs-getting-started.spec.ts

# Architecture

## Project Structure
- ./projects
    - ./accion-labs
            - ./home.page.po.ts
    - ./playwright-dev
        - ./docs-page
            - page objects by section
